# Email Rules

## Greensky (GreenSky Credit/Loan Program)

### Detection
- **From domain**: greenskycredit.com, greensky.com
- **Subject keywords**: Greensky, GreenSky, GSKY
- **Known senders**: noreply@greenskycredit.com, resources@greenskycredit.com

### Handling
1. **Label**: Apply the `Greensky` label
2. **Archive**: Remove from INBOX (keep labeled for reference)
3. **Mark Read**: After review, mark as read

### Action Items

#### "Transaction Request Authorized By Customer"
- Customer has approved the financing transaction
- Note the customer name, Application ID, and amount
- Check if 50% deposit has been submitted (coordinate with sales/office)

#### "GREENSKY WO [number] / Work Order threads"
- WO needs to be processed in the system
- Actions typically required:
  - **50% deposit**: Submit deposit request via GreenSky portal
  - **Quote/Work Order update**: Update pricing/WO in company system
  - Confirm submission with team (typically Aaron Senseney)

#### Password/Maintenance Emails
- Password changes/reset confirmations are informational only
- No action required unless unauthorized

### Coordination
- Mario Umbach typically initiates WOs (Outside Sales)
- Aaron Senseney handles GreenSky submissions
- William submits deposits and quote updates
