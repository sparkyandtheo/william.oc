How to install in Gmail

  1. Click Copy signature above.
  2. In Gmail, click the gear icon → See all settings.
  3. Scroll to Signature → click + Create new (or pick an existing one).
  4. Click inside the signature edit box and paste with Ctrl+V (Windows) or Cmd+V (Mac).
  5. Set "Signature defaults" so it appears on new emails & replies, then Save Changes at the bottom.
